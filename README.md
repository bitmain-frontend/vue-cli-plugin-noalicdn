# A vue-cli plugin to check if there is alicdn css font in .js and .vue files

## Install

```vue add @bitmain/noalicdn```

## Usage

### options

```
--input the directory where the raw .js&.vue files is in(default: src)

```
#### by vue-cli-service
```javascript
  npm run noalicdn // or
  npx vue-cli-service noalicdn
```
